Beans Reels

Fab Pavilion

A fab showroom pavilion on a budget for Seoul MakerFaire 2015

Semi inflatable installation where the "hard"(wood ribs) represent the reality,the making,the hardware and the "soft"(ballons) represent the intangible,the ideas,the imagination.


The space

We get to make 50 square meters of building completely made with digital fabrication tools.(Standart shopbot cnc) The total cost was 3000$ and it is reusable as only used wood joinery "no screws,no glue" for the assembly.

  1. Serve as a showroom of the FabLab capabilities
  2. Portable workshop space
  3. Host some talks
  4. Access an main gate of Seoul Maker Faire

Fabrication

It is cnc cutted frame with inflatable ballons between each rib. On each rib there is a Led RGB stripe that it is controlled with an arduino and fab made pcb drivers boards.The "control panel" is made in acryl with buttons so people can interact with the lighting(Laser cutted)All the furniture it is also cnc.
